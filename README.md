## Requirements

## Installation

```bash
$ docker run -p 5000:5000 -e PORT=5000 -d raphaelbh/timezone-api
```
    
## Usage

```bash
$ curl http://localhost:5000/api
```

## Running Tests

```bash
$ pytest --cov=app tests/
```
